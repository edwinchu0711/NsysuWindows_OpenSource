## 功能說明 (Features)

* **選課助手系統**：支援使用者自訂預排課程，並提供將預排名單直接匯出至學校選課系統的功能。
* **課表管理與匯入**：提供直觀的課表介面，並支援快速匯入現有課表資料。
* **成績與名次預覽**：自動爬取歷年成績，提供成績列表與搶先名次預覽（僅供參考）。
* **數據自動同步**：採用增量更新技術，每次登入自動同步「近期」課表與成績，兼顧效能與流量。
* **網路大學整合**：可直接存取網路大學的作業、考試與公告，並設有數據保護機制避免流量異常。
* **智慧學期切換**：內建自動化引擎，隨學年更迭（2月/8月）自動切換顯示數據，確保時效性。
* **異常處理功能**：於特定選課時段，App 內會顯示按鈕供下載並產出「異常處理單」。
* **畢業審核參考**：提供畢業審核資訊概覽，幫助學生掌握學分修習進度。
* **重要日程提醒**：整合校方行事曆。

## 安裝與下載 (Installation)

如欲使用請至對應之專案頁面下載：

* **Android / iOS (App)**：請至 **[NsysuApp_OpenSource](https://github.com/edwinchu0711/NsysuApp_OpenSource)** 下載對應檔案。
    * **Android**：下載 `.apk` 檔案即可直接安裝。
    * **iOS**：下載 `.ipa` 檔案，須透過 **Sideload (側載自簽)** 方式進行安裝（如 AltStore 或 SideStore）。
* **Windows**：請至  **[Releases 頁面](../../releases)** 下載安裝程式。
